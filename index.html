4 · Render capability — Invoke-AliceImageGen.ps1 (the only code that touches the API)
RESOLVE KEY
Read credentials
Environment/.env → $env:OPENAI_API_KEY fallback → throw with setup-doc pointer.
BUILD REQUEST
JSON body
model=gpt-image-2, prompt, size, quality, n=1. Bearer auth header.
POST
OpenAI Images API
POST https://api.openai.com/v1/images/generations
DECODE + SAVE
b64 → PNG
Decode data[0].b64_json, create parent dir if missing, write bytes to OutFile.
🔒 Security boundary (Mandatory Rule 16): the API key is read inside the helper and never returned. On any error the message is scrubbed — sk-... patterns are replaced with [REDACTED-KEY] before the exception is thrown. The key never appears in logs, diary, commits, or chat.

Returns (PSCustomObject)

saved_path · cost_estimate_usd · elapsed_seconds · model · size · quality — Prism reports cost to Kiyo every render (Rule 19, renders never feel “free”).
